# Eventify Frontend

## Overview

This directory contains the frontend implementation of **Eventify**, a web-based event management platform developed as part of an MSc Computer Science dissertation project.

The frontend provides the user interface of the platform and enables users to interact with the system through a responsive web application. The platform is designed to support event discovery, event creation, organiser services, and event planning activities.

## Installation

Install project dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build the application for production:

```bash
npm run build
```
