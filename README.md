Smart Event Management Platform
Overview

This project aims to develop a Smart Event Management Platform that assists users in discovering, creating, and managing events through a unified web-based system. The platform integrates event management, service booking, organiser recommendations, visual event customisation, and communication functionality within a single environment.

The project is being developed as part of the MSc Computer Science dissertation.

Features

The current scope of the project includes:

User registration and authentication
Event creation and management
Organiser page management
Service booking
Planning assistant
Visual simulation tool
Natural language search
Chat system
Reminder system
Multi-language support (English and Arabic)
